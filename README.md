How to Play...

Use the arrow keys to move your paddle

Stop the ball from touching the purple or green line on your side.

If it does, you can click restart (the "r" key)

You can always pause it by pressing the p key.

There are 2 dev profiles, dev and admin. When you put them in, it will ask for a password. Try to guess it😊😊. The dev accounts have crazy effects  which makes it very hard.

There is an AI account where, if you create an account named AI, it will assign an AI bot to you. The password is AI.  You have to score on it.

All the highscores save even if you close the game.
